# awg-bot — журнал изменений

## v1.2.2 — объявления пользователям, самопроверка старта

> ℹ️ Патч поверх v1.2.1: новая рассылка объявлений, самодиагностика при запуске и мелкие правки интерфейса. Отдельных действий при установке не требуется.

### 📢 Новое

- **Объявление пользователям.** Кнопка «📢 Объявление пользователям» в админ-меню: пишешь текст (можно с разметкой — <b>жирный</b>, ссылки), он уходит всем клиентам и активным друзьям, по одному разу на человека. Перед отправкой — превью и число получателей, после — отчёт о доставке.
- **Самопроверка при запуске.** Бот проверяет окружение на старте: критичные проблемы (повреждённая БД, нет места для записи, отвергнутый токен) останавливают запуск с понятным сообщением в журнале; некритичные (мало места на диске, не отвечает контейнер, недоступен IMAP аварийного выхода) приходят тебе первым сообщением после запуска.

### ✨ Интерфейс

- В списках устройств тип помечается иконкой: 🕹 — полный доступ к серверу, 📲 — устройство друга, 📱 — обычное.
