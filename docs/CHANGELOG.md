# awg-bot — журнал изменений

## v1.2.0 — Настройки прямо в чате (горячая конфигурация)

> ℹ️ **Важно:** появился раздел **⚙️ Настройки** в админ-меню — ходовые параметры меняются кнопками и применяются на горячую, без рестарта. В главном меню больше нет «Бэкап», «Обновление бота», «Перезапустить AWG» — всё переехало в Настройки. Источник истины прежний — `conf/*.yaml`; правка руками работает так же горячо (за каталогом следит вотчдог). Добавлена зависимость `ruamel.yaml` (ставится автоматически при обновлении). После обновления перезапусти бота один раз, дальше рестарты для смены настроек не нужны.

### ⚙️ Экран настроек

- Новый раздел **⚙️ Настройки** — шесть экранов, значения меняются кнопками/вводом и применяются сразу:
  - 🔔 **Уведомления**: тихие часы (окно без звука), алерты хоста (CPU/RAM/диск), четыре независимых тумблера событий клиентов (активация, самопродление, превышение лимита, выдача бонуса);
  - 💳 **Параметры подписок**: бонус-квота, макс. дней паузы, grace-дней;
  - 📊 **Мониторинг**: частота опроса, порог стрика, громкость и порог алерта простоя AWG;
  - 💾 **Резервное копирование**: день и час автобэкапа, «бэкап сейчас»;
  - 🔄 **Сервис**: перезапуск AmneziaWG и самого бота;
  - ⬆️ **Обновления бота**: тумблер уведомлений, расписание проверки (час/день/неделя/месяц/никогда), «проверить сейчас».
- Ввод значений — с проверкой диапазонов; «расписание: никогда» автоматически выключает уведомления об обновлениях.

### 🔥 Горячая конфигурация

- Параметры читаются из `conf/*.yaml` в момент использования — правка (из чата или руками) применяется без рестарта. За каталогом следит вотчдог; изменения применяются точечно, только к затронутым параметрам.
- Расписания планировщика (частота монитора, автобэкап, проверка обновлений, очистка истории) перевешиваются на лету.
- Запись из чата сохраняет комментарии и порядок в yaml-файлах. Повреждённый файл не роняет бота — прежние значения остаются в силе, в лог пишется предупреждение.
