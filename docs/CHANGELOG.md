# awg-bot — журнал изменений

## v1.2.1 — доработки интерфейса настроек

> ℹ️ Патч поверх v1.2.0: правки интерфейса. Отдельных действий при установке не требуется.

### ✨ Интерфейс

- Кнопки подключения в главном меню — «🔗 Ссылка», «🔳 QR-код», «📄 Файл» — компактнее, в один ряд.
- Настройки → раздел «Сервис» переименован в «🔄 Обслуживание».
- Пороги CPU/RAM/диск в алертах хоста — одной строкой.
- Расписание проверки обновлений: убран вариант «каждый час» (осталось день/неделя/месяц/никогда); при наличии старого значения оно автоматически переносится в «каждый день».
- Гайд Apple (создание казахстанского Apple ID): формат подсказок выровнен на «English (Русский)».
